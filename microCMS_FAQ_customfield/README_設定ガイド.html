microCMS カスタムフィールド（繰り返し）FAQ 設定ガイド

1. microCMS管理画面でのAPI設定

1 対象のAPIスキーマ編集画面を開く
2 「フィールドを追加」→ 種類：繰り返し を選択
3 繰り返しフィールドの設定：
項目設定値
フィールドIDfaq
表示名よくあるご質問
4 繰り返しフィールドの中に、以下2つのサブフィールドを追加：
フィールドID表示名種類
question質問テキストフィールド
answer回答リッチエディタ または テキストエリア
補足：回答に太字・リンクなどを使いたい場合は「リッチエディタ」を選択してください。プレーンテキストで十分な場合は「テキストエリア」でOKです。

2. APIレスポンスの形式

上記設定をすると、APIから以下のようなJSONが返ります：

{
  "id": "article-001",
  "title": "記事タイトル",
  "faq": [
    {
      "fieldId": "faqItem",
      "question": "サービスの利用に必要なものはありますか？",
      "answer": "インターネット環境とメールアドレスがあれば..."
    },
    {
      "fieldId": "faqItem",
      "question": "料金プランについて教えてください",
      "answer": "無料プランと有料プランをご用意しております..."
    }
  ]
}

3. フロント側の実装

同梱の faq-accordion.html をご利用ください。設定箇所は2つだけです：

// ===== ここを書き換えてください =====
const SERVICE_DOMAIN = 'your-service-id';  // microCMSのサービスID
const API_KEY        = 'your-api-key';     // APIキー
const ENDPOINT       = 'blogs';            // エンドポイント名
const CONTENT_ID     = 'your-content-id';  // 記事のコンテンツID
// =====================================
