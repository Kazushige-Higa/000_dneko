クライアント様のご紹介 設定手順

ブログ・制作実績・コラムの各APIに繰り返しフィールド clients を追加し、記事内にクライアント情報をカード形式で紹介できるようにします。

完成イメージ：

Sample Cafe

沖縄県宜野湾市○○1-2-3

海が見えるおしゃれカフェ。自家焙煎コーヒーとスイーツが人気です。

営業時間: 10:00〜18:00 / 定休日: 火曜
詳しくみる

STEP 1：繰り返しフィールド「clients」を追加

1 microCMS管理画面 → 対象API（blog / works / column）→ 右上「API設定」→「APIスキーマ」タブ
2 「フィールドを追加」→ 繰り返しフィールド（カスタムフィールド） を選択
項目設定値
フィールドIDclients
表示名クライアント様のご紹介
種類繰り返しフィールド（カスタムフィールド）
必須いいえ

STEP 2：繰り返しフィールド内のサブフィールドを設定

3 繰り返しフィールド「clients」の中に以下のサブフィールドを追加

基本情報

フィールドID表示名種類必須備考
nameクライアント名テキストフィールドいいえ店名・会社名など
address住所テキストフィールドいいえ例: 沖縄県宜野湾市○○1-2-3
image画像画像いいえロゴ・外観写真など
description紹介文テキストエリアいいえクライアントの説明（改行可）
freetext自由欄テキストエリアいいえ営業時間・定休日など何でも

「詳しくみる」ボタン

フィールドID表示名種類必須備考
detail_url詳細リンクURLテキストフィールドいいえ自由なURL（公式サイト、マップ、予約ページなど何でもOK）
detail_label詳細リンクの表示名テキストフィールドいいえ未設定時は「詳しくみる」と表示
ボタンの表示名をカスタマイズできます。例: 「公式サイトはこちら」「Googleマップで見る」「ご予約はこちら」など。detail_label を空欄にすると「詳しくみる」がデフォルト表示されます。

SNS・各種リンク

フィールドID表示名種類必須アイコン
url_websiteWebサイト URLテキストフィールドいいえ fa-globe
url_instagramInstagram URLテキストフィールドいいえ fa-instagram
url_lineLINE URLテキストフィールドいいえ fa-line
url_xX (Twitter) URLテキストフィールドいいえ fa-twitter
url_facebookFacebook URLテキストフィールドいいえ fa-facebook
url_youtubeYouTube URLテキストフィールドいいえ fa-youtube
url_tiktokTikTok URLテキストフィールドいいえ fa-tiktok
url_otherその他 URLテキストフィールドいいえ fa-external-link-alt
url_other_labelその他URLの表示名テキストフィールドいいえホバー時のツールチップに使用。未設定時は「その他リンク」
ポイント：すべてのフィールドは任意（必須：いいえ）です。使わないフィールドは空欄のままでOK — 空欄のフィールドは表示されません。SNSのURLフィールドは必要なものだけ入力してください。
重要：3つのAPI（blog / works / column）すべてに同じ clients 繰り返しフィールドを追加してください。サブフィールドのIDも同じにすることで、PHPコード側で共通の表示処理が使えます。

STEP 3：記事でクライアント情報を入力

4 各記事の編集画面で「クライアント様のご紹介」欄の「＋追加」ボタンでクライアントを追加

入力例：

フィールド入力例
クライアント名Sample Cafe
住所沖縄県宜野湾市○○1-2-3
画像店舗の外観写真
紹介文海が見えるおしゃれカフェ。自家焙煎コーヒーとスイーツが人気です。
自由欄営業時間: 10:00〜18:00 / 定休日: 火曜
詳細リンクURLhttps://sample-cafe.com/
詳細リンクの表示名公式サイトはこちら
Instagram URLhttps://www.instagram.com/samplecafe/
LINE URLhttps://line.me/R/ti/p/@xxx
補足：繰り返しフィールドなので、1つの記事に複数のクライアントを追加できます。「＋追加」を押すたびにクライアントカードが増えます。

スキーマ完成イメージ

blog / works / column API（既存）
├─ title, content, thumbnail ... ← 既存フィールド
├─ category（セレクト）← カテゴリタブ用
├─ faq（繰り返し）← FAQ
├─ cta（繰り返し）← CTAボックス
├─ speech（繰り返し）← 吹き出し
│
└─ clients（繰り返しフィールド）← 今回追加
    ├─ name（テキスト）← クライアント名
    ├─ address（テキスト）← 住所
    ├─ image（画像）← ロゴ・写真
    ├─ description（テキストエリア）← 紹介文
    ├─ freetext（テキストエリア）← 自由欄
    ├─ detail_url（テキスト）← 詳しくみるリンクURL
    ├─ detail_label（テキスト）← 詳しくみるボタンの表示名
    ├─ url_website（テキスト）← Web
    ├─ url_instagram（テキスト）← Instagram
    ├─ url_line（テキスト）← LINE
    ├─ url_x（テキスト）← X (Twitter)
    ├─ url_facebook（テキスト）← Facebook
    ├─ url_youtube（テキスト）← YouTube
    ├─ url_tiktok（テキスト）← TikTok
    ├─ url_other（テキスト）← その他URL
    └─ url_other_label（テキスト）← その他表示名

表示の仕組み

条件動作
clients にデータありCTAボックスの後、ライターカードの前にクライアントカードを表示
clients 未設定何も表示されない
画像未設定画像エリアなしでテキストのみ表示
SNS URL未設定そのSNSアイコンは非表示
detail_url 設定あり「詳しくみる」ボタンを表示（detail_label で表示名変更可）
detail_url 未設定ボタン非表示
自由欄設定ありベースカラーのサイドラインが入ったボックスで表示
